# Python Collections HackerRank-Style Practice Course

GitHub-ready medium-to-complex coding problems focused on Python collections.

## Topics Covered

- `list`, stack patterns, sorting, sliding windows
- `dict`, nested dictionaries, indexes, joins
- `set`, intersections, differences, graph visited sets
- `tuple`, sort keys, immutable composite keys
- `collections.Counter`, `defaultdict`, `deque`, `OrderedDict`
- `heapq` priority queues
- Mixed real-world data engineering and healthcare/insurance scenarios

## How to Use

1. Read each problem in `problems/`
2. Implement or review the matching solution in `solutions/`
3. Add tests in `tests/`
4. Push this folder to GitHub
