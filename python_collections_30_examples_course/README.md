# Python Collections HackerRank-Style Course

This GitHub-ready package contains 30 commented Python examples per collection: 10 simple, 10 medium, and 10 complex.

Collections covered: list, tuple, set, dict, Counter, defaultdict, deque, heapq.

Run any file directly with `python filename.py`.

## list
- ordered, mutable sequence
- Folder: `lists/`

## tuple
- ordered, immutable record/sequence
- Folder: `tuples/`

## set
- unique unordered collection
- Folder: `sets/`

## dict
- key-value mapping
- Folder: `dictionaries/`

## collections.Counter
- frequency map
- Folder: `counter/`

## collections.defaultdict
- dictionary with automatic default values
- Folder: `defaultdict/`

## collections.deque
- fast queue/stack from both ends
- Folder: `deque/`

## heapq
- priority queue / top-k structure
- Folder: `heapq/`
